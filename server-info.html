Purpose: Design, implementation, and ongoing maintenance of personal home server using Proxmox VE hypervisor on laptop hardware (Intel i7-7500U, 16GB RAM, 1TB SSD). Deployed multiple services through Virtual Machines and LXC containers including Home Assistant OS for home automation, Docker containerization with Karakeep for bookmark management, Pi-hole for network-wide ad blocking, Homepage dashboard for centralized service access, Beszel for system monitoring, and Ntfy for server notifications. Secured remote access implemented via Tailscale. This project demonstrates practical knowledge of virtualization, containerization, networking, and system administration.
